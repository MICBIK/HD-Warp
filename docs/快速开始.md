# 🚀 HD Warp 快速开始指南

本指南帮助您在 5 分钟内完成 HD Warp 的安装和配置。

---

## 📋 前置准备

### macOS 用户

打开「终端」应用，依次执行以下命令：

```bash
# 1. 安装 Homebrew（如已安装可跳过）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 2. 安装 Python 3.13+
brew install python@3.13

# 3. 安装 mitmproxy（二选一）
brew install mitmproxy
# 或
pip3 install mitmproxy
```

### Windows 用户

1. 下载并安装 [Python 3.10+](https://www.python.org/downloads/)
   - ⚠️ 安装时勾选「Add Python to PATH」
2. 打开 CMD，执行：
   ```cmd
   pip install mitmproxy
   ```

---

## 📥 安装应用

1. 前往 [Releases](../../releases) 下载对应系统的安装包
2. **macOS**: 双击 `.dmg` 文件，将应用拖入「应用程序」文件夹
3. **Windows**: 双击 `.exe` 安装程序，按提示完成安装

### ⚠️ macOS "应用已损坏" 解决

如果提示应用已损坏，打开终端执行：

```bash
sudo xattr -rd com.apple.quarantine /Applications/HD\ Warp.app
```

---

## 🔐 安装证书

证书是网关正常工作的关键，请务必完成此步骤。

1. 启动 HD Warp
2. 进入「网关中心」页面
3. 点击「生成证书」按钮
4. 点击「立即安装」
5. 在弹出的钥匙串窗口中，**双击证书** → **信任** → 选择「始终信任」

> 💡 验证：在「钥匙串访问」中搜索 `mitmproxy`，确认证书显示为蓝色 ✓

---

## 👤 添加账号

1. 进入「账号池」页面
2. 点击「添加账号」按钮
3. 在弹出框中输入 API Key（每行一个）

**API Key 格式示例**：
```
wk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
wk-yyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyy
```

**如何获取 API Key**：
1. 打开 Warp IDE
2. 进入 Settings → Account
3. 点击「Create API Key」
4. 复制生成的 Key

---

## ✅ 启动网关

1. 进入「网关中心」页面
2. 点击「输入激活码」，输入 16 位激活码
3. 点击「启动网关」
4. 等待左下角显示「就绪账号」数量
5. 正常打开 Warp 使用即可！

---

## 🎉 完成！

现在您可以正常使用 Warp 的 AI 功能了。网关会自动：

- ✅ 替换请求中的 API Key
- ✅ 监控账号额度使用情况
- ✅ 额度耗尽时自动切换账号
- ✅ 检测并跳过被封禁的账号
